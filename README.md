# Sales Team CRM (CLI Application)

A Python command-line application that manages sales team performance.

## Features

- Add salesperson
- Update salesperson data
- Delete salesperson
- Search salesperson
- Generate team reports
- Calculate sales statistics

## Technologies Used

- Python
- Dictionaries
- Nested Data Structures
- CLI (Command Line Interface)

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run:

python main.py

## Example

Sales Team CRM

1 Add salesperson  
2 Update/Delete salesperson  
3 Reports & Statistics  
4 Exit
